Preparation
-----------

To run the `buildings.py` script you must download the [TIGER data](https://www2.census.gov/geo/tiger/TIGER2023/COUNTY/tl_2023_us_county.zip) from the US Census Bureau and extract the following files:

- `tl_2023_us_county.dbf`
- `tl_2023_us_county.shp`
- `tl_2023_us_county.shx`
